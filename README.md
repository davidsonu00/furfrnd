# Furfrnd

Furfrnd is a full-stack dog selling web application with separate user and admin flows.

## Features

### User
- Register from home page
- Receive OTP
- Verify OTP
- Login
- Land on shop page

### Admin
- Login from home page
- Land on admin dashboard
- Create products

## Tech Stack
- Node.js
- Express.js
- MongoDB
- EJS
- JWT
- Bootstrap / CSS

## Project Flow
- User lands on home page, registers, gets OTP, verifies OTP, logs in, and visits shop page.
- Admin lands on home page, logs in, reaches admin dashboard, and creates products.

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Create a `.env` file using `.env.example`
4. Run `npm start`

## Environment Variables
See `.env.example` for required keys.
